## Quick start:

```bash
# installing dependencies
$ npm install

# testing/development
$ npm run dev

# building/production
$ npm run build
```

in case you use `Yarn`:

```bash
# installing dependencies
$ yarn

# testing/development
$ yarn dev

# building/production
$ yarn build
```
